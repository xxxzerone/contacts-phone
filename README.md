# 연락처 목록 가져오기 앱
시스템 연락처 정보를 가져와서 보여준다.

- 연락처 정보를 가져와서 리스트로 표시하고 초성 검색 기능을 구현
- 특정 연락처를 즐겨찾기 하여 DB에 저장
